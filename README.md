# 🌿 Wild Life Vista  
### Wildlife Tracking & Analytics Platform  

---

## 📌 Project Overview  
**Wild Life Vista** is a static web-based application designed to simulate a **wildlife tracking and monitoring system**.  

It helps visualize animal movement, detect critical events, and provide analytics for forest officials and researchers.  

This project demonstrates how modern web technologies can be used to build an **interactive dashboard for wildlife conservation**.

---

## 🎯 Objectives  
- Track animal movements using simulated GPS data  
- Detect critical events like:  
  - Zone breach  
  - Inactivity  
  - Abnormal speed  
- Provide visual dashboards and reports  
- Implement role-based access (Admin / Ranger)  

---

## 🧩 Modules  

### 🔐 User Management  
- Login system (Admin / Ranger roles)  
- Dynamic dashboard access  

### 🐾 Animal Tracking  
- Simulated real-time tracking  
- Google Maps integration  

### 🚨 Alerts & Notifications  
- Zone breach detection  
- Inactivity alerts

### 📊 Analytics & Reports  
- Movement analytics (Chart.js)

## ▶️ How to Run the Project  
1. Download or clone the project  
2. Open `index.html` in any browser  
3. Select role (Admin / Ranger)  
4. Click **Login**  
5. Explore dashboard features
